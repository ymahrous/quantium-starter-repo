# Soul Foods - Pink Morsels Sales Visualiser

## Overview
This Dash app visualises daily sales of **Pink Morsels** and allows filtering by region. Users can explore trends over time and see the impact of events such as price changes.  

**Features:**
- Interactive line chart of daily sales
- Region filter (North, East, South, West, All)
- Price increase annotation
- Styled layout for a better user experience
